HTML
Hyper Text Markup Language
Much clearer, right?
No
Lets break it down
Markup language = Document + Tags
Basically synonymous with
adding notes to a document
In other words
Marking it up
(ah-ha)
So what are tags?
Tags are composed of
Start tag
<tagname>
End tag
</tagname>
And something in the middle
(mostly)
Multiple sets are nested
YES:
<A> <B> </B> </A>
NO:
<A> <B> </A> </B>
Browsers interpret these
tags to figure out how to
display content in
an HTML document
There are tags for
all kinds of stuff
images
links
structure
and much more

But still rather abstract
